# Installation

    cd ~/Library/Application\ Support/TextMate/Bundles
    svn co http://macromates.com/svn/Bundles/trunk/Bundles/ANTLR.tmbundle

# Post-install

If [TextMate](/wiki/TextMate) is running, choose "Reload Bundles" from the "Bundle Editor" submenu in the "Bundles" menu.
