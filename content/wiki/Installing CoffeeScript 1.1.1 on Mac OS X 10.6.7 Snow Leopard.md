We install via [npm](/wiki/npm), so we have a couple of prerequisites:

-   [node.js](/wiki/node.js): see "[Installing node.js 0.4.7 on Mac OS X 10.6.7 Snow Leopard](/wiki/Installing_node.js_0.4.7_on_Mac_OS_X_10.6.7_Snow_Leopard)"
-   [npm](/wiki/npm): see "[Installing npm 1.0.6 on Mac OS X 10.6.7 Snow Leopard](/wiki/Installing_npm_1.0.6_on_Mac_OS_X_10.6.7_Snow_Leopard)"

The actual installation is simple:

```shell
$ npm install -g coffee-script
```
