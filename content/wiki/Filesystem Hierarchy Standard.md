The [FHS](/wiki/FHS) or [Filesystem Hierarchy Standard](/wiki/Filesystem_Hierarchy_Standard) is:

> A set of requirements and guidelines for file and directory placement under UNIX-like operating systems. The guidelines are intended to support interoperability of applications, system administration tools, development tools, and scripts as well as greater uniformity of documentation for these systems.
