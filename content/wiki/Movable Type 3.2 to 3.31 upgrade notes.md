# Upgrade steps

-   (For the sensible) backup all Movable Type files and the database
-   Connect to `mt-check.cgi` to ensure that minimum Perl version (5.6.1) requirement is met
-   Upload new files
-   Hit `mt-upgrade.cgi`
-   Delete `mt-upgrade.cgi`
