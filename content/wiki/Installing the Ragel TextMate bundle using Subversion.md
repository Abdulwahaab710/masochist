# Installation

    cd Library/Application\ Support/TextMate/Bundles
    svn co http://macromates.com/svn/Bundles/trunk/Bundles/Ragel.tmbundle

# Reloading

Quit and relaunch [TextMate](/wiki/TextMate), if it is running (choosing "Reload Bundles" from the "Bundle Editor" submenu in the "Bundles" menu is not sufficient if this is the first time you've installed the bundle).
