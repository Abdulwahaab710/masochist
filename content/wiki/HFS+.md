[HFS+](/wiki/HFS%2b) is an infamously unreliable [Apple](/wiki/Apple) filesystem which has caused countless [Mac OS X](/wiki/Mac_OS_X) users no end of grief. Unlike most serious [UNIX](/wiki/UNIX)Â filesystems, HFS+ is not case-sensitive by default (although it can optionally be made so), and it performs a lossy Unicode "normalization" on paths which can cause multiple headaches when working in a cross-platform environment.
