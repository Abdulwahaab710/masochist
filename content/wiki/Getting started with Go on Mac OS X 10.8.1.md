1.  Install binary installer from <http://golang.org/doc/install>
2.  Set up the [Vim](/wiki/Vim) go files; eg. <http://git.wincent.com/wincent.git/commit/6c1685ae016b551c6096dd836f51d20f44fcc12d>
3.  Set up `~/.vimrc` settings; eg. <http://git.wincent.com/wincent.git/commit/b71620b1620893341bc582d8469086aa52e87add>
4.  `sudo go get -u github.com/nsf/gocode`
5.  `git clone git://github.com/nsf/gocode.git`
6.  `cd gocode/vim`
7.  `./pathogen_update.bash`
